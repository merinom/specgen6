Abstract

Starfire is a standardized data schema (also referred as "ontology" or "vocabulary") designed to use for diagnosis failures in a network and try to solve them. The following document contains the description of ontology and instructions how to connect it with descriptions of other resources.

1 Introduction

The following specification is a formal description of metadata schema proposal that can be applied to diagnosis failures in a network and try to resolve the failure according to the diagnostic. The goal of the following section is to introduce goals of the ontology and provide the basic knowledge to comprehend the technical part of the specification.

1.1 What is Starfire for?

The goals of the Starfire Ontology to achieve as a data schema are:

  • make a Diagnosis for failures that happen in a Network

  • try to solve the failures according the diagnosis.
